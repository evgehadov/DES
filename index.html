■ DES CIPHER

Data Encryption Standard — ECB / CBC
Как пользоваться: выберите операцию (Encrypt / Decrypt), режим (ECB или CBC), введите текст и ключ ровно 8 символов → нажмите Run.
Для дешифрования вставьте hex-результат в поле Input и переключите операцию на Decrypt. В режиме CBC скопируйте Used IV из вывода и вставьте его при дешифровании.
Операция
Режим
ECB — простой режим, каждый блок шифруется независимо
Формат ввода
Text — обычный текст  |  Hex — шестнадцатеричный
Формат вывода
Для шифрования рекомендуется Hex
Входные данные
Ключ — ровно 8 символов
0/8
Введите ровно 8 символов — не больше, не меньше
IV — вектор инициализации (16 hex-символов)
При дешифровании вставьте сюда значение «Used IV» из результата шифрования
Результат
Used IV: ← скопируйте для дешифрования
Состояние по раундам — первый блок (сеть Фейстеля, 16 раундов)
#L (32 бит)R (32 бит)Подключ (48 бит)
